# Brahma Context Engine

AI-powered pattern recognition and privacy-first consciousness scaffolding

## Features

- **Glassmorphism UI** - Beautiful, modern interface with blur effects and transparency
- **Context Capture** - Multi-layer thought capturing (Surface, Systemic, Legacy, World)  
- **Voice Entry** - Speech-to-text integration for natural thought capture
- **AI Insights** - Pattern recognition and insight generation
- **Privacy-First** - All data stored locally, no external servers
- **Real-time Stats** - Live tracking of notes, insights, and active layers
- **Export Functionality** - Save your contexts as JSON, Markdown, or text

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Development

- Built with React 18 + Vite
- Styled with Tailwind CSS
- Glassmorphism effects with custom CSS
- Local storage for data persistence

## Integration Notes

- **Voice Recording**: Connect your speech-to-text service in `VoiceEntryModal.jsx`
- **AI Insights**: Integrate your AI service in `InsightPreview.jsx`  
- **Social Features**: Placeholder components ready for collaboration features
